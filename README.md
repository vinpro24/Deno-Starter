# Deno-Starter
Deno - A secure runtime for JavaScript and TypeScript.

## Installation

```
git clone https://github.com/vinpro24/Deno-Starter.git
npm install
```

## Config

```
.env
/contants/config.js
```

## Run on development

```
deno run --allow-run --allow-net --allow-read --allow-env --unstable app.ts
denon start
```
### Run with Denon
```
denon start
```

## Build production

```
deno build
```
